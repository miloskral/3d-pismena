Features

Everything you need

Six dedicated panels — every parameter a sign maker needs, nothing more.

📂

DXF Import

Load any vector DXF file directly from your CAD or illustration tool. Choose unit scale (millimeters or inches) and set curve tolerance down to 0.05 mm for highly precise geometry. The app immediately shows the file size and the number of curves detected.

Unit scale Curve tolerance Multi-shape support
🔮

Real-time 3D Preview

See your letter update in 3D as you change any parameter. Switch between Isometric, Top, and Front views. Toggle between Wireframe and Acrylic render modes. The header always displays live triangle count, exact dimensions, and shape count.

Iso / Top / Front Wireframe & Acrylic Live triangle count
🧱

Wall Settings

Set the outer wall thickness (default 2 mm) and box depth (default 25 mm). A built-in cross-section diagram shows exactly how the wall, acrylic panel, and tolerances relate to each other.

Wall thickness Box depth Cross-section diagram
💎

Acrylic & Rebate

Configure acrylic thickness (3 mm), rebate width (1 mm), clearance around the acrylic panel (0.2 mm), and the inset below the top edge. All tolerances are tuned for a precise press-fit with standard acrylic sheet thicknesses.

Acrylic thickness Rebate width Clearance control
🔩

Back Cover System

W – Open (no back cover), I – Solid (flat closed back), S – Snap-in (snap lip for tool-free acrylic installation). Configure snap lip height (6 mm), lip thickness (1.6 mm), an optional cable hole, and back-to-wall clearance (0.3 mm).

Open / Solid / Snap-in Cable hole Snap lip geometry
🎨

Color System

Choose wall body color and translucent acrylic color independently. Eight presets each, plus a full color picker for any RAL or custom hex value. An opacity slider lets you simulate how the finished letter will look when illuminated.

Wall & acrylic colors Custom hex picker Opacity simulation
📦

Dual STL Export

Download two separate files: the Wall STL (the letter body with the acrylic rebate) and the Back STL (the back cover or snap-in lid). Both files are FDM print-ready — no further preparation in a slicer is required before printing.

Wall STL Back STL FDM-ready
🔑

Offline License

Your license key is stored locally on your machine — no account required, no server calls, no data sent anywhere. Paste your key once and STL export is permanently unlocked. The app works fully offline, including on air-gapped production computers.

No account required Works offline Local key storage

How it works

From DXF to STL in 3 steps

01

Import your DXF file

Load a vector file from any CAD or illustration tool. Set the unit scale and curve tolerance. The app immediately shows the bounding-box dimensions and the number of shapes detected.

02

Configure all parameters

Adjust wall thickness, box depth, acrylic rebate, back cover type, snap lip geometry, cable hole, and wall and acrylic colors — all with live 3D feedback in the viewport.

03

Generate and download the STL files

Click Generate 3D Model, then download the Wall STL and Back STL separately. Both files are ready to slice and print on any FDM printer. The license is verified locally — no internet connection required.
